Set Logic: a set is a group of objects, elements, or members represented as a unit, the set may be finite or infinite, the order of the objects is not pertinent. The character “∈” is used to denote if an object membership to a set. E.G. √2∈ℝ, but ℚ∌√2 is read “√2 is an element in the set of real numbers, but the set of rational numbers does not contain the element √2.
Important Sets:
  1. ∅, denoting the empty set: ∅ = {}.
  2. ℙ, denoting the set of all primes: P = {2, 3, 5, 7, 11, 13, 17, ...}.
  3. ℕ, denoting the set of all natural numbers: N = {1, 2, 3, . . .} (sometimes defined containing 0).
  4. ℤ, denoting the set of all integers (whether positive, negative or zero): Z = {..., −2, −1, 0, 1, 2, ...}.
  5. ℚ, denoting the set of all rational numbers (that is, the set of all proper and improper fractions): Q = {a/b : a, b ∈ Z, b ≠ 0}. For example, 1/4 ∈ Q and 11/6 ∈ Q. All integers are in this set since every integer a can be expressed as the fraction a/1 (Z ⊊ Q).
  6. ℝ, denoting the set of all real numbers. This set includes all rational numbers, together with all irrational numbers (that is, numbers that cannot be rewritten as fractions, such as √2, as well as transcendental numbers such as π, e and numbers that cannot be defined).
  7. ℂ, denoting the set of all complex numbers: C = {a + bi : a, b ∈ R}. For example, 1 + 2i ∈ C.
  8. ℍ, denoting the set of all quaternions: H = {a + bi + cj + dk : a, b, c, d ∈ R}. For example, 1 + i + 2j − k ∈ H.


Sequences and Tuples: a sequence is a set in which order is pertinent. Finite sequences often are called tuples. A sequence with k elements is a k-tuple.
(Undir/Dir)ected Graphs: pair notation: the set of vertices or nodes followed by the set of (edges, lines, arcs, or arrows) that connect the nodes.

Fallacies:
  1. Modus Ponens: A → B; A; ∴ B;
  2. Modus Tollens: A → B; ¬ B; ∴ ¬ A;
  3. Generalization: A; ∴ A ∨ B: B; ∴ A ∨ B;
  4. Specialization: A ∧ B; ∴ A and ∴ B;
  5. Conjunction: A; B; ∴ A ∧ B;
  6. Elimination: A ∨ B; ¬ B; ∴ A;
  7. Transitivity: A → B; B → r; ∴ A → r;
  8. Division into Cases: A ∨ B; A → r; B → r; ∴ r;
  9. Contradiction Rule: ¬ A → ⊥; ∴ A;
